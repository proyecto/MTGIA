# Pages Documentation

### `Dashboard.tsx`
The landing page. Shows a summary of the collection:
- Total value.
- Recent additions.
- Quick stats.
- News or updates (if implemented).

### `Collection.tsx`
The main view for browsing the user's collection.
- Supports filtering and sorting.
- Displays cards in a grid.
- Clicking a card opens the `CardDetailsModal`.

### `Sets.tsx`
Allows browsing cards by set.
- Lists all available sets.
- Clicking a set shows cards within that set.

### `Wishlist.tsx`
Displays the user's wishlist.
- Allows adding/removing items.
- Shows target prices and priorities.

### `Settings.tsx`
User configuration:
- Currency preference (USD/EUR).
- Data management (Import/Export).
