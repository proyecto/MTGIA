# MTG Collection Manager Documentation

Welcome to the documentation for the MTG Collection Manager application.

## Structure

This documentation is divided into two main sections:

- **[Backend Documentation](./backend/README.md)**: Details about the Rust/Tauri backend, including database operations, services (Scryfall), and commands.
- **[Frontend Documentation](./frontend/README.md)**: Details about the React/TypeScript frontend, including components, pages, and state management.

## Getting Started

To run the application in development mode:

```bash
npm run tauri dev
```

To build the application:

```bash
npm run tauri build
```
